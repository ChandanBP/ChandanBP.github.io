Natural Language Processing

This repository contains natural language processing techniques implmented in java. Ngram modelling for predicting next words in the sequence of a given input sentence is implemented using bigram modelling. Backward viterbi algorithm is used to predict the hidden state weather sequence for the given HMM(Hidden Markov Model). Context sensitive meanings for the given words in the input sentence is implemented using lesk and simple lesk algorithms. Context free spell check is implemented similar to messenger, search engine query is implemented using dynamic programming approach with the suitable heruristics to trim the searhc space and optimal solution is implemented.
